# Telemost Hotkeys (KTalk-like)

Юзерскрипт:
1. Добавляет в Яндекс Телемост привычные горячие клавиши из Контур.Толка.
2. Подсвечивает активные кнопки: микрофон, камеру, поднятую руку и демонстрацию экрана.

## Установка

1. Установите расширение [Tampermonkey](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) для Chrome ([для других браузеров](https://www.tampermonkey.net/))
2. Откройте [скрипт](https://raw.githubusercontent.com/happyproff/userscript-telemost-hotkeys/main/telemost-hotkeys.user.js) и нажмите Install
3. Если Телемост уже открыт, обновите страницу

## Горячие клавиши

| Сочетание | Действие                                |
|-----------|-----------------------------------------|
| Alt+**A** | Включить / выключить микрофон           |
| Alt+**V** | Включить / выключить камеру             |
| Alt+**R** | Поднять / опустить руку                 |
| Alt+**D** | Начать / остановить демонстрацию экрана |
| Alt+**C** | Открыть / закрыть чат                   |

## Обновление

Если вы устанавливали скрипт через Tampermonkey, обновления будут предлагаться автоматически.
